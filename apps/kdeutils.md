# KDEUTILS

 This is the KDE utils suite into one AppImage.

 Usage:

 - kdeutils PROGRAM
 - kdeutils PROGRAM OPTIONS

 This scripts includes icons and launchers for the following programs:

 - ark : Archiving tool for .zip/.tar/.rar and more. This script installs the full "kdeutils" suite.
 - francis : Pomodoro technique, get more productive. This script installs the full "kdeutils" suite.
 - isoimagewriter : Write an ISO Image to a USB Disk. This script installs the full "kdeutils" suite.
 - kalm : Teach you different breathing techniques. This script installs the full "kdeutils" suite.
 - kate : A multi-document editor part of KDE. This script installs the full "kdeutils" suite.
 - kbackup : Back up your data. This script installs the full "kdeutils" suite.
 - kcalc : Scientific Calculator. This script installs the full "kdeutils" suite.
 - kcharselect : Character Selector. This script installs the full "kdeutils" suite.
 - kclock : A universal clock application. This script installs the full "kdeutils" suite.
 - kdebugsettings : Edits displayed QLoggingCategory. This script installs the full "kdeutils" suite.
 - kdf : view Disk Usage. This script installs the full "kdeutils" suite.
 - kdialog : a dialog utility. This script installs the full "kdeutils" suite.
 - keditbookmarks : Bookmarks editor. This script installs the full "kdeutils" suite.
 - keysmith : Generates tokens for your two-factor logins. This script installs the full "kdeutils" suite.
 - kfind : A search tool. This script installs the full "kdeutils" suite.
 - kgpg : A GnuPG frontend. This script installs the full "kdeutils" suite.
 - kongress : A companion application for conferences. This script installs the full "kdeutils" suite.
 - krecorder : An audio recording application. This script installs the full "kdeutils" suite.
