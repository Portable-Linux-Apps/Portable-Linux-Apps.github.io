# NODE

 This scripts installs the following programs:

 - node : Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine.
 - npm : The package manager for the Node JavaScript platform. It puts modules in place so that node can find them, and manages dependency conflicts intelligently.
 - npx : This command allows you to run an arbitrary command from an npm package (either one installed locally, or fetched remotely), in a similar context as running it via "npm run".
