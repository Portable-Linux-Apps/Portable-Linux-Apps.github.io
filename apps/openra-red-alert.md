# OPENRA-RED-ALERT

 Open Source real-time strategy game engine for early Westwood games such 
 as Command & Conquer: Red Alert written in C# using SDL and OpenGL. Runs 
 on Windows, Linux, *BSD and Mac OS X.
 
 This is the MOD "Red Alert"
