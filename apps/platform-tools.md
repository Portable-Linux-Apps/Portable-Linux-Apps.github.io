# PLATFORM-TOOLS

 Android SDK Platform Tools is a complete suite of Command line utilities needed to manage Android devices.

 This scripts installs the following programs:

 - adb : Android Debug Bridge. A command-line tool for communicating with an Android device or emulator. Commonly used for installing apps, viewing logs (logcat), running shell commands, transferring files, debugging, and managing devices over USB or TCP/IP.
 - etc1tool : Utility for working with ETC1 texture compression files used in graphics/OpenGL ES on Android. It can compress images into ETC1 format or decompress ETC1 textures back into standard image formats.
 - fastboot : Low-level Android bootloader tool used to flash partitions, unlock bootloaders, reboot devices, and recover or install firmware images. Works when a device is in fastboot/bootloader mode.
 - hprof-conv : Converts Android HPROF heap dump files into standard Java HPROF format so they can be analyzed with Java memory analysis tools like Eclipse MAT or VisualVM.
 - make_f2fs : Creates an F2FS (Flash-Friendly File System) filesystem on a partition or image file. F2FS is optimized for NAND flash storage such as eMMC and SSDs, commonly used on Android devices.
 - make_f2fs_casefold : Variant/helper related to creating F2FS filesystems with casefolding support enabled. Casefolding allows case-insensitive filename matching while preserving original filename case.
 - mke2fs : Creates ext2/ext3/ext4 Linux filesystems on partitions or image files. Often used when formatting disks or generating filesystem images for embedded/Linux/Android systems.
 - sqlite3 : Command-line interface for SQLite databases. Lets you create, inspect, query, and modify SQLite database files using SQL commands or interactive shell features.
